# What is the expected behavior of the repo?

### discord-forwarder

this plugin is used to forward message from one channel to another channel in discord, it should work like this:
1. user/bot send a message in the source channel, the message can be text, image, video, file or embed
2. the plugin receive the message, and forward the message to the target channel, the forwarded message should be in the same format as the original message, and the forwarded message should also indicate who is the original sender, and the forwarded message should also indicate the original channel
